# Workflow: Rapid Client Pitch Generation

Follow these steps to generate a high-end, SEO-optimized, white-labeled landing page for a client in 10-12 minutes.

## 1. Flexible Project Scope
- Build time and complexity adapt to client needs (Target range: 10-20 mins for pitches).
- Gather: Business Name, Category, Rating, Reviews, Address, Phone.

## 2. Visual Asset Generation
- Generate high-resolution AI images tailored to the business category.
- Match the "Vibe" identified during discovery.

## 3. Custom UI Design
- **No Fixed Default**: Typography and color palettes must be chosen based on the specific project and client requirements.
- Aim for premium aesthetics that suit the category (e.g., minimalist for tech, luxury for salons).
- Interactions: Implement animations that enhance the specific branding.

## 4. Professional SEO Injection
- **JSON-LD**: Always include LocalBusiness schema.
- **Semantic HTML**: Use header, nav, main, section, footer tags.
- **Meta Tags**: Optimized description, keywords, and Open Graph tags.

## 5. WHITE-LABELING MANDATE
- **CRITICAL**: Never include "Built by Antigravity" or any AI signature.
- **Footer**: Use "Crafted with Precision" or simple copyright notice.

## 6. Instant Deployment
- Use `deploy.py [client_folder]` to push to GitHub Pages immediately.
- Verify the live link before sharing.
